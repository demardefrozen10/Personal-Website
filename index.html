Latest Project

IronStrong

A full web application that helps avid gym goers search for specific exercises based on the exercise name or muscle. Gives precise instructions on how to perform the exercise, the difficulty level, and the equipment required to perform such exercises. Also, a warm-up page where users can enter their desired weight for their working set, and it calculates how many sets, reps, and weight they should preform in-order to properly warmup for their exercise.

Technologies used include:

  • HTML
  • CSS
  • Python
  • Bootstrap
  • Flask
  • BeautifulSoup
  • Jinja

Deng Finance

An informational website in an attempt to improve teenagers financial literacy. With over 75% of Canadians in debt, it is no secret that schools and teachers lack the knowledge to teach such a subject. At dengfinance.ca, teenagers can learn the step by step blueprint on how to become a millionare by simply investing $42 per day in the S&P 500. It also demonstrates the various accounts that can be opened in Canada for tax advantages.

Technologies used include:

  • HTML
  • CSS
  • WordPress
  • Elementor
